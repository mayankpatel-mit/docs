## Get All Live Surveys

Retrieve a list of all live surveys based on filtering criteria.

### Endpoint
```http
POST /surveys
```

### Headers
```json
{
  "publisher-authentication-key": "your_project_secret_key"
}
```

### Request Body
```json
{
  "authorizationKey": "DqcInvhIFAl5WtUNwihFUIgUH77EL2hTNRU",
  "country": "US",
  "language": "English",
  "lengthOfInterviewGTE": 1,
  "lengthOfInterviewLTE": 5,
  "collectPII": false,
  "CPIGTE": 1,
  "CPILTE": 5,
  "surveyGroupExists": false,
  "incidenceRateGTE": 1,
  "incidenceRateLTE": 25
}
```

### Response
```json
{
  "success": true,
  "surveys": [
    {
      "surveyNumber": 100000001,
      "surveysId": "encryptedSurveyId",
      "loi": 5,
      "ir": 25
    }
  ],
  "surveysCount": 1,
  "projectName": "Testing project",
  "projectNumber": 100000001,
  "organisation": "mirats insights"
}
```

### Error Response
```json
{
  "message": "internal server error",
  "code": 500,
  "success": false
}
```
